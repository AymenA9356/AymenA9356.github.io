making walter white good Project

Context

Mr. Mueller, my SEP 10 teacher, assigned me and my partner David to create our own "Adventure" by using our prior knowledge of what we were taught on the last two subjects which was Git and GitHub webpage.The title of our adventure is Making walter White good.

Content

We have to creat a project that uses what we have learned so far in linking files. We were givien the option to link the files with HTML(Hypertext Markup Language) or we can use markdown whitch is language that is used with plain text to add formatting elements (headings, bulleted lists, URLs). Me and my partner went with markdown, and linking the fiels using our IDE. to link the fiels because it looked easy enough to do so insted of linking with HTML. The requirements for this project was to have at least 10 files,a story linked to them, and to have plan on what is going to be linked.

Self-Reflection

The most challenging I have gone through was having too many merge conflicts. So communication is very important in these kinds of situations. Another challenge that we faced was not being able to link the files proporly, and we kept having a 404 error code. We had to be very creative because we had to come up this from scratch. In the end communication and collaboration is key for great teamwork. Preview of the Project
